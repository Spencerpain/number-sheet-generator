# Number Sheet Generator

A simple tool to generate printable number fill-in-the-blank sheets.

**Live Site:** https://spencerpain.github.io/number-sheet-generator

## Features

- 10-wide number grid counting from any start to any end number
- 3 difficulty levels: Easy, Medium, Hard
- Print a single sheet or mass generate 10, 15, 20, or a custom number of pages
- Print-ready — controls are hidden when printing
